## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites
* NPM
* React Router 5
* Dayjs



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/justincanthony/travel-tracker-remix.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
